Wire It Together in Jenkins

    Go to your Jenkins Dashboard and select New Item -> Pipeline. Name it hello-swarm-pipeline.

    Under Credentials -> System -> Global credentials, 
    click Add Credentials. Choose Username with password, fill in your Docker Hub credentials, 
    and name the ID docker-hub-credentials-id.

    In your Pipeline job configuration, scroll down to Pipeline, 
    change the definition to Pipeline script from SCM, choose Git, and provide your repository URL.

    Click Save and click Build Now.
After configuration when all the swarm int commands are pasted and joined then:-

Verify Nodes (Manager Terminal)
docker node ls

Create a service:
docker service create \
--name spring-service \
-p 8080:8080 \
lucifer1124/spring-app:latest

Check Running Services
docker service ls

Check Tasks
docker service ps spring-service

Check Nodes
docker node ls

Inspect the Service
docker service inspect spring-service

Scale the Application
docker service scale spring-service=3
